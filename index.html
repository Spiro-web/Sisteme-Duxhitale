Pajisje Kombinatore — Half-Adder, Full-Adder, 4-bit Paralel

Ç'janë Mbledhësit?

Mbledhësi (Adder) është një qark logjik kombinatorial që kryen mbledhjen aritmetike të bitave binare. Ai është blloku themelor i njësisë aritmetiko-logjike (ALU) në çdo procesor.

// Llojet kryesore:
Half-Adder = Mbledh 2 bita hyrëse (pa bartje hyrëse)
Full-Adder = Mbledh 3 bita (me bartje hyrëse Cin)
4-bit Adder = 4× Full-Adder të lidhur në seri (ripple carry)

Half-Adder — Mbledhësi i Thjeshtë

Half-Adder mbledh dy bita hyrëse (A dhe B) dhe prodhon Shumën (S) dhe Bartjen (C). Nuk ka bartje hyrëse — prandaj quhet "gjysëm mbledhës".

// Shprehjet Boolean:
S = A ⊕ B   // XOR — Shuma
C = A · B   // AND — Bartja

Full-Adder — Mbledhësi i Plotë

Full-Adder mbledh tre bita: A, B dhe bartjen hyrëse Cin. Ndërtohet nga dy Half-Adder + një portë OR.

// Shprehjet Boolean:
S = A ⊕ B ⊕ Cin
Cout = (A · B) + (Cin · (A ⊕ B))

4-bit Ripple Carry Adder

Katër Full-Adder të lidhur në seri (zinxhir). Bartja Cout e çdo stadeje bëhet Cin i stades tjetër. Mund të mbledhë numra deri në 15 + 15 = 30 (4-bit).

// Struktura:
FA₀: A₀+B₀+0 → S₀, C₀
FA₁: A₁+B₁+C₀ → S₁, C₁
FA₂: A₂+B₂+C₁ → S₂, C₂
FA₃: A₃+B₃+C₂ → S₃, Cout ← overflow!
